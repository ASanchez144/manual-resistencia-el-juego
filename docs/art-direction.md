# Dirección artística

## Estilo base

- Ilustración 2D tipo cartoon/vectorial.
- Líneas limpias y marcadas.
- Personajes de cuerpo completo.
- Fondos verde chroma key `#00FF00` cuando el asset necesite recorte.
- Assets pensados para animación en videojuego 2D.

## Nomenclatura

Usar nombres descriptivos y versionados:

```text
assets/sprites/characters/<personaje>/<personaje>_<accion>_<version>.png
```

Ejemplo:

```text
assets/sprites/characters/presidente/presidente_run_cycle_001.png
```

## Primer personaje

Personaje principal con traje azul marino, camisa blanca y corbata. Estilo político caricaturesco, elegante y reconocible sin depender de realismo fotográfico.
